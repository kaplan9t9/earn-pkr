# Earn in Rupees PK

Pakistan's trusted earning platform where users can earn by watching ads.

## Features
- 3 Premium Plans (₨300, ₨500, ₨1000)
- Watch 20 ads daily
- Deposit via Easypaisa/JazzCash
- Admin approval for deposits and withdrawals
- Admin dashboard with user management

## Tech Stack
- Frontend: HTML + Vue.js + Tailwind CSS
- Backend: Netlify Functions (Node.js)
- Database: MongoDB Atlas
